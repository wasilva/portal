Para configuração inicial do projeto:

1-Instalar o node.js link: https://nodejs.org/en/
2-Instalar o mongoDB caso seja o banco a ser utilizado link: https://www.mongodb.com
3-No prompt de comando acesse o diretório do projeto e de um npm init, para iniciar o npm.
4-Instale as dependecias do projeto: npm install <dependecias> --save 
    express
    express-validator
    express-session
    ejs
    nodemon
    mongodb
    mysql
    consign
    socket.io (chat)
